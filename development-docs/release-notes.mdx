Rolling log of all changes done on the system.

### 2023/05/29 - QoL Updates
* Updated `SystemOptics` and `SystemComms` to allow more visibility and reception of messages while in Offline
* Updated matrix connected to include Region name
* Updated Jolt/Teleport particles to use texture instead of default particle

### 2025/04/20 - RELEASE
**25.4.19.1 – Rubber Kingdom Controller Release**
- **New RK1 Controller:** Released in partnership with Rubber Kingdom, this new controller model introduces modular upgrades, improved power management, and optimized particle effects handling.
- Added the `AbstractPlasmaController`, designed to improve code reuse and simplify future controller designs.
- Refactored media menu system to allow HUD-based media prim adjustments using the `|hud menu` command.
- Minor updates to the ProjectArousal plugin for compatibility.

**25.3.21.1 – Plugin Refactor and Setting Copy Support**
- Standardized method declarations across devices and items, improving plugin logic consistency, especially in dialog handling and attachment behavior.
- Adjusted thermal stat behavior and updated dependency versions.
- Introduced setting copy support for major version upgrades, allowing users to easily transfer settings between controller generations.

**25.4.19.2 – Power System Overhaul**
- Introduced the `AbstractSpringController` to centralize power handling logic.
- Replaced older NexusController logic with modular, reusable methods.
- Improved item initialization via `on_item_created` in PowerSpringDevice.

**25.4.20.2 – Filter & Persona Fixes**
- Refactored dialog button filters to use `setInclude`, improving selection reliability.
- Added error logging for better debugging support.
- Resolved issue where persona and filter notecards were not appearing in dialog selection.
- Updated dependency versions.

### 2025/03/11 - BUG-FIXES
- Quick deployment to fix some bad user experiences.
- Adding fix for minor bug within ACS service during charging.
- Removing 'Fluids' button from Manage menu.

### 2025/03/07 - RELEASE
This update introduces new API functionality to support recently released products like the new display case and chargers. 
- Power level warnings now trigger at 25% increments for better monitoring. 
- RLV handling has been refactored to improve action processing, such as sitting.
- Fixed tesseract bug for users who have no teleport history.
- Added new functions for KORI API.
- Releasing KOR's Display Case.
- Releasing a HEV HL themed plasma charger.

### 2025/01/10 - RELEASE

- Refactored notecard handling and improved regex functionality.
- Fixed closed button on the UI.

### 2025/01/06 - BREAK-FIX
- Pushing out an update to fix the controller erroring out after running the `!update` command.

### 2025/01/04 - Retro Controllers & Bug Fixes
- Bugfixes on `KOR.comms`, `KOR.devices`, `KOR.probe`, and `KOR.collide` to fix stack-heaping.
- Two new electric-based controllers released with a retro vibe.
- Refactored MediaModule and collision handling.
- Introduced a comprehensive logging service.
- Enhanced fluid management (rates, disposal, conversion).
- Improved power management and API support.
