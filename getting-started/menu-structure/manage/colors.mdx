---
title: "Colors"
---

All units have 4 color channels. You can change these channels through the menu: **Menu > Manage > Color**.

Alternatively, you can use the command: <code>!color &lt;CHANNEL&gt; &lt;HEX, RGB, or COLOR NAME&gt;</code>

### Color Channels

| Channel | Description |
| :--- | :--- |
| **A** | Primary Unit Color |
| **B** | Secondary Unit Color |
| **C** | Tertiary Unit Color |
| **D** | Quaternary Color |

<Note>
  All new KOR users start with 'red' as their primary color.
</Note>
