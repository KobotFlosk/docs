---
title: "RLV Relay"
---

This menu allows the unit full control over their RLV Relay.

## Relay
Toggles RLV relay for the controller, **ON** or **OFF**.

## Access
Access mode determines how the unit interacts with incoming RLV requests.

| Mode | Description |
| :--- | :--- |
| **Public** | No prompts for access, auto capture. |
| **Simple** | Prompts for access by Prim. |
| **Advanced** | Prompts for access and allows granting by Prim, Name, or Region. |

## Safeword
Set your own safe word.

## Permissions
Lets you manage RLV relay permissions by Name, Object, Region, Owner, or Creator. You can see a list of things you added and remove any relay permissions you have given.
