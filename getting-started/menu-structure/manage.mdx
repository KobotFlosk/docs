---
title: "Manage"
---

The **Manage** menu allows you to personalize your unit and configure core operational toggles.

### Menu Options

| Option | Description |
| :--- | :--- |
| **Bolts** | A toggle option. **ON** locks the controller and attached devices. **OFF** allows them to be removed while running. |
| **[Users](/getting-started/menu-structure/manage/users)** | Sub-menu for User/Owner Configurations and access levels. |
| **[RLV Relay](/getting-started/menu-structure/manage/rlv-relay)** | Manage RLV Relay permissions and RLV settings. |
| **[Designation](/getting-started/menu-structure/manage/designation)** | Change the unit's **NAME** or **LABEL**. |
| **Gender** | Set the unit's gender identity. |
| **[Color](/getting-started/menu-structure/manage/colors)** | Set the unit's primary and custom color channels (A, B, C, D). |
| **HERO** | Toggle the [H.E.R.O Combat System](/secondary-systems-optional/hero-combat-system) on/off and display status. |
