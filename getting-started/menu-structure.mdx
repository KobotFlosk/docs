This is KOR's Menu Structure, with detail.

<Note>
This menu is shown by either running the command **!menu** OR **|menu** OR **clicking** the back of the controller.
</Note>

## Main Menu

| Menu Option | What does it do? |
| :--- | :--- |
| **[Subsystem](/getting-started/menu-structure/subsystems)** | Gives you access to the units Subsystems Configurations |
| **[Core](/getting-started/menu-structure/core)** | This is where you can configure the units mind, and memory states |
| **Resources** | Links to our Discord, SL Group, and basic help! |
| **[Follow](/getting-started/menu-structure/follow)** | Search the nearby area for agents within 5m to instruct the unit to follow |
| **[Devices](/getting-started/menu-structure/devices)** | See the units current connected devices (e.g., Batteries, Chargers) |
| **[Report](/getting-started/menu-structure/report)** | Reports the current system status (Power, Integrity, etc) into local chat |
| **[Manage](/getting-started/menu-structure/manage)** | Change the units color, gender, and name. Personalize your unit! |
| **Shutdown** | Turns the unit off |
