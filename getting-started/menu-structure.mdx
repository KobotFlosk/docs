This is KOR's Menu Structure with detailed descriptions of each option.

<Note>
This menu is shown by either running the command `!menu` OR `|menu` OR clicking the back of the controller.
</Note>

## Main Menu

| Option | Description |
| :--- | :--- |
| **[Subsystems](/getting-started/menu-structure/subsystems)** | Gives you access to the unit's Subsystems Configurations. |
| **[Core](/getting-started/menu-structure/core)** | Configure the unit's mind, memory, filters, and personas. |
| **Resources** | Links to our Discord, SL Group, and basic help! |
| **[Follow](/getting-started/menu-structure/follow)** | Search nearby for agents within 5m to instruct the unit to follow. |
| **Devices** | See the unit's current connected devices (e.g., Batteries, Chargers). |
| **Report** | Reports the current system status (Power, Integrity, etc.) into local chat. |
| **[Manage](/getting-started/menu-structure/manage)** | Change the unit's color, gender, and name. Personalize your unit! |
| **Shutdown** | Turns the unit off. |
