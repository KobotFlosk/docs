title: "KOR Overhead"
---

The KOR Overhead comes included with every purchase of a KOR system.

To use it, simply attach the overhead device to the **Nose**. This innovative overhead text device is the first to offer gradient colors.

This device enables the display of visual statistics above a unit's head.

### Overhead Values

| Value             | What does it mean?                                                 |
| ----------------- | ------------------------------------------------------------------ |
| `[designation]`   | Units NAME or LABEL                                                |
| `[gender]`        | Units gender                                                       |
| `[power:level]`   | Units current charge level                                         |
| `[power:rate]`    | Units current power draw                                           |
| `[shield]`        | If a shield is attached it will show the current % of shield level |
| `[stat:integrity]`| Units current chassis health                                       |
| `[temperature]`   | Units current temperature                                          |
| `[following]`     | Shows who the unit is currently following                          |
| `[mood]`          | Shows the units current set mood                                   |
| `[arousal]`       | *Not yet implemented*                                              |

### Overhead Template Example

To personalize your Overhead Device, navigate to the [Devices](/getting-started/menu-structure) menu and choose '**KOR Overhead**'. Upon clicking '**Title**', a dialog will appear showing your current title, allowing you to make modifications.

<Frame caption="Title Dialog">
  <img src="/images/gitbook/overhead-2.png" />
</Frame>

Below is a sample of the standard overhead text. Feel free to modify or expand upon it as needed.

```text
[designation]
[gender] | [power:level] | [power:rate]
[shield] | [stat:integrity] | [temperature]
```

```text
[designation]
Sex: [gender] | [power:level] | [power:rate]
[mood] | [stat:integrity] | [temperature]
[following]
```

### Overhead Gradient Colors

To customize the colors of the Overhead, access the device by going to the [**Devices**](/getting-started/menu-structure) menu and selecting **KOR Overhead**.

In the menu, select '**Colors**' to add up to *10 colors*, and each added color will contribute to creating a gradient effect across each line currently displayed by the device.

<Note>
