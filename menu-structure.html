üìÑMenu Structure

This is KOR's Menu Structure, with detail.

This menu is shown by either running the command !menu OR |menu OR clicking the back of the controller.

Main Menu

Menu Option
What does it do?

Subsystem

Gives you access to the units Subsystems Configurations

Core

This is where you can configure the units mind, and memory states

Resources

Links to our Discord, SL Group, and basic help!

Follow

Search the nearby area for agents within 5m to instruct the unit to follow

Devices

See the units current connected devices

Report

Reports the current system status into local chat

Manage

Change the units color, gender, you name it! Personalize your unit

Shutdown

Turns the unit off

Restart

Soft-resets the system (No longer an option)
