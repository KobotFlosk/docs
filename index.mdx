## Overview

KOR is a thin-client based RLV Controller for Second Life. The system addresses LSL limitations to provide a more reliable and stable experience.

KOR is a line of Robot/Synthetic RLV Controllers that provides improved speed and reliability.

## Key Points

- **Over-the-air updates**: Bug-fixes and updates deploy automatically without user intervention.
- **100% Thin-client**: The controller communicates with an external back-end for seamless operation in Second Life.
- **Automatic updates**: System updates occur automatically without manual hardware changes.
- **High-performance**: Lightweight scripts provide superior performance and stability. Compatible with nearly all products.
- **Compatibility**: Approximately 95% compatible with all Nanite Systems products.

KOR includes an RLV Relay that retains your RLV settings across different sims and furniture.
